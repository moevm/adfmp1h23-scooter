# Установка  
Перейти по [ссылке](https://yandex.ru/dev/maps/mapkit/?from=mapsapi) и выпустить токен яндекс MapKit SDK  
```
cd app/src/main/assets/
cp env.template env
```
Отредактировать файл `env` и вставить свой токен от MapKit SDK
